1. 아이디어 <br>
그래프를 그린다 [[0], [1번노드와 연결된 노드들], [2번노드와 연결된 노드들], ...] <br>
BFS를 돌리고 카운트 + 1 <br>
연결된 노드들 chk True로 바꿔준다 <br>
모든 노드에 대하여 실행 <br><br>

2. 자료구조 <br>
chk<br>
graph <br>
BFS: queue <br>

<br>

```
import sys
from collections import deque

def BFS(start):
    queue = deque()
    queue.append(start)
    chk[start] = True

    while queue:
        num = queue.popleft()
        for i in graph[num]:
            if chk[i] == False:
                queue.append(i)
                chk[i] = True


input = sys.stdin.readline

N, M = map(int, input().split())

graph = [[0] for _ in range(N+1)]
chk = [False] * (N+1)
cnt = 0

# 그래프 그리기
for _ in range(M):
    u, v = map(int, input().split())
    graph[u].append(v)
    graph[v].append(u)

for i in range(1, N+1):
    if chk[i] == False:
        BFS(i)
        cnt += 1

print(cnt)
```


### 오답노트
- 그래프 그리는 법을 몰랐음
- `graph = [[0] for _ in range(N+1)]`
- 노드 수보다 하나 더 크게 만든다. (0번 인덱스는 비워 둔다)
- 각 인덱스는 각 노드를 의미하며, 리스트 안에 들어 있는 내용은 그 노드와 연결된 노드들을 의미한다
- 그래프 그리는 법
  
  ```
  for _ in range(M):
    u, v = map(int, input().split())
    graph[u].append(v)
    graph[v].append(u)
  ```
