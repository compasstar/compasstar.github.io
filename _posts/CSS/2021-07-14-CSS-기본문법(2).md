# 7. 개체 속성

***
```
<head>
    <style>
        .one{
            box-sizing: border-box;
            border: solid 20px black;
            padding:20px;
            width:200px;
            height:200px;
            background-color:red;
            opacity: 0.3;
        }
        .two{
            border: outset 20px gray;
            border-radius: 20px;
            margin:20px;
            width:200px;
            height:200px;
            background-color:green;
        }
        .three{
            border: solid 20px black;
            width:200px;
            height:200px;
            background-color:blue;
        }
    </style>
</head>
<body>
    <div class="one">hello world</div>
    <div class="two"></div>
    <div class="three"></div>
</body>
```

![css2](https://user-images.githubusercontent.com/55419868/125565152-9947b3fa-f7c4-48a6-8a38-fb1b7412569a.JPG)


***
 
`border`
> 테두리를 조절할 수 있습니다.  
> 테두리의 종류로는 `solid`, `dotted`, `double`, `dashed`, `groove`, `ridge`, `inset`, `outset` 들이 있습니다.


`box-sizing`
> 박스의 크기를 고정시킵니다.  
> 첫번째 박스와 세번째 박스를 비교해봅시다.  
>  세번째 박스는 테두리로 인해 크기가 240x240이지만, 첫번째 박스는 크기가 고정되어 200x200 입니다.

`border-radius`
> 모서리를 둥글게 합니다.

`opacity`
> 투명도를 뜻합니다.   
  0(투명)~1(불투명) 까지 설정할 수 있습니다.

# 8. 위치

***

```
<head>
    <style>
        .container{
            width:600px;
            height:600px;
            background-color:skyblue;
            position: absolute;
            top: 10px;
            left: 10px;
        }
        .one{
            width:100px;
            height:100px;
            background-color:red;
            position: absolute;
            margin: -50px;
            top: 50%;
            left: 50%;
        }
        .two{
            width:100px;
            height:100px;
            background-color:blue;
            position: absolute;
            top: 30px;
            left: 30px;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="one"></div>
        <div class="two"></div>
    </div>
</body>
```

![css3](https://user-images.githubusercontent.com/55419868/125567589-a8153c33-e358-4c59-97d3-c0d359b1b8fe.JPG)



***

`absolute`
> 부모요소를 기준으로 위치를 설정합니다.  
> 대비되는 특성으로 `relative`가 있는데, 이것은 자신의 본래 위치(static) 기준으로 위치를 설정합니다.  
> 만일 `.one` 과 `.two`의 위치를 동일하게 설정한다면 둘은 겹치게 됩니다.


가운데정렬
>`.one`은 현재 가운데 정렬이 된 상태입니다.   
`top 50%`, `left 50%` 로 인해 왼쪽상단 꼭짓점이 `.container`의 중앙으로 오게 됩니다.  
크기가 100x100 이므로 `margin`을 -50으로 설정하여 중앙으로 오게 했습니다.<br><br>
만일 inline 속성을 가졌다면, 가운데 정렬이 더 쉽습니다.  
`margin: 0, auto;`  
상하 여백은 0이고, 좌우 여백은 자동으로 맞추어져 가운데 정렬이 됩니다.

<br>
아래 이미지와 같이 한 줄을 통째로 차지하는 속성을 <strong>inline</strong> 이라고 합니다.

# 9. 폰트

***

```
<head>
    <link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Nanum+Brush+Script&display=swap" rel="stylesheet">
    <style>
        #font  {
            font-family: 'Nanum Brush Script', cursive;
            font-size: 32px;
            }
    </style>
</head>
<body>
    <div id="font">hello world</div>
</body>
```

# 10. 색깔

***

```
<head>
    <style>
        #color{
            width:100px;
            height:100px;
            background-color:#0000ff;
        }
    </style>
</head>
<body>
    <div id="color"></div>
</body>
```

<head>
    <style>
        #color{
            width:100px;
            height:100px;
            background-color:#0000ff;
        }
    </style>
</head>
<body>
    <div id="color"></div>
</body>

***

색깔은 `red`, `skyblue` 이런 식으로 표현할 수도 있지만, `#000000` 형태로도 표현할 수 있습니다.

구글에 'color picker' 혹은 'color code' 라고 검색하면 컬러코드를 얻을 수 있습니다.

개발자도구로 들어가도 컬러코드가 나옵니다.

![css7](https://user-images.githubusercontent.com/55419868/125571980-d1421bd2-8d79-4ada-acde-7219d585db00.JPG)


`#000000` 6자리는 각 2자리씩 RGB를 나타냅니다.  
`#ff0000`는 Red이고, `#00ff00`는 Green, `#0000ff`는 
Blue입니다.  
`#000000`은 Black,  `#ffffff`은 White입니다.
