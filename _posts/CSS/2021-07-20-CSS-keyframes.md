`displey: flex`에 관하여 알아보겠습니다.

```
display: flex;
justify-content: center;
flex-direction: column;
```

`display: flex;` 
> flex container를 정의해줍니다. <br>
이제부터 다른 속성들을 사용할 수 있습니다.

`justify-content: center;` 
> 정렬방법을 설정하는 것으로, `center`를 통해 가운데 정렬을 할 수 있습니다.

`flex-direction: column;`
> 주 축의 방향을 결정해줍니다. <br>
>  default 값은 `row`입니다. <br>
> `column`으로 설정하면 객체들이 세로방향으로 나열됩니다.
