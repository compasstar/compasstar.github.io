```
text-decoration: underline;
```
텍스트에 밑줄을 긋습니다.

```
text-decoration: none;
```

텍스트에 부과된 효과를 없앱니다. <br>
주로 `a` 태그와 같이 밑줄이 기본으로 그어져있는 경우 사용됩니다.
