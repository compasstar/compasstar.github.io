페이지를 꾸밀 때, 기본 형식이 정해져 있습니다. <br>

예를들어 Excel에서의 기본폰트는 맑은고딕이고, 글자크기는 11p입니다. <br>

css를 통해 style을 꾸며야 하는데, 자신이 원하지 않는 style이 적용되어 있다면 코드를 짤 때 불편할 것입니다. <br>

그러므로 style을 초기화 시키고 css코드작성을 시작하면 편리합니다.

이때 필요한 것이 <strong>css reset</strong> 입니다.


<br>
<br>


![css reset](https://user-images.githubusercontent.com/55419868/126271995-02085d20-eeeb-4394-991e-384f0735c17d.JPG)

reset.css 파일을 만들어 그곳에 css reset 코드를 붙여주시면 됩니다.

css reset 코드는 
<a href="https://meyerweb.com/eric/tools/css/reset/">https://meyerweb.com/eric/tools/css/reset/</a>
이곳에서 가져올 수 있습니다. 구글에 css reset이라고 검색하셔도 나올 것입니다.


```
<link rel="stylesheet" type="text/css" href="css/reset.css">
```
그 후, html 파일의 `<head>`에 위의 코드를 통해 css파일과 연결해주시면 됩니다.
