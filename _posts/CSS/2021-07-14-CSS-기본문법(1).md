CSS는 페이지를 꾸미는 역할을 합니다. 예를들어 글씨 색을 바꾼다거나, 가운데 정렬을 한다거나 할 수 있습니다.

구름 IDE를 이용했음을 미리 밝힙니다.

단축키는 명령어를 입력하고 Tap을 입력하면 됩니다. 

> h + Tap --> `<h1></h1>`

<br>
<br>

# 1. CSS 코드 입력하기

***

<h1 style="color:red;">hello world</h1>

hello world를 붉은색으로 꾸미는 코드를 어디에 입력해야 할까요?


```
<head>
    <style>
        h1{color:red;}
    </style>
</head>
<body>
    <h1>hello world</h1>
</body>
```
첫번째 방법은 HTML파일의 `<head>`에 `<style>`로 작성하는 것입니다. 

```
<h1 style="color:red;">hello world</h1>
```
두번째 방법은 `<h1>`에 직접 `<style>`을 적어주는 것입니다.
```
<head>
    <link rel="stylesheet" href="001.css">
</head>
<body>
    <h1>hello world</h1>
</body>
```
```
h1{color:red;}
```
세번째 방법은 .css파일을 따로 만들어 link를 걸어주는 것입니다.  
위의 코드는 .html 파일이고, 아래 코드는 .css 파일입니다.  
두 파일을 link로 연결시켜주면 됩니다.  
단축키는 `link` + `Tap` 입니다.  

<br>
앞으로는 첫번째 방법을 사용해서 설명하겠습니다.


<br><br>

# 2. 명칭

***
```
<style>
    h1, h2, h3{
        color:skyblue;
    }
</style>
```
<h1 style="color:skyblue;">hello world</h1>
<h2 style="color:skyblue;">hello world</h2>
<h3 style="color:skyblue;">hello world</h3>

***

`<style>`을 기준으로  
h1, h2, h3 을 <strong>선택자(Selector)</strong>라고 합니다.  
color 을 <strong>속성(Property)</strong>라고 합니다.  
skyblue 을 <strong>값(Value)</strong>라고 합니다.



<br><br>

# 3. 선택자

***

```
<head>
    <style>
        * {font-weight: bold;}
        h2 {font-size: 4px;}
        .one {color:red;}
        .two {border: solid 1px black;}
        #three {font-style: italic;}
    </style>
</head>
<body>
    <h1 id="three">hello world</h1>
    <p>hello world</p>
    <p class="one two">hello world</p>
    <p class="one">hello world</p>
    <h2>hello world</h2>
</body>
```

![css1](https://user-images.githubusercontent.com/55419868/125548557-6bf9f5bc-1f92-41e9-b882-27b0ebb2ba2f.JPG)

***

`*`
> <strong>범용선택자</strong>로, 모두에게 적용시킵니다.
  
`h2`
><strong>타입선택자</strong>로, h2에게 적용시킵니다.

`.one`, `.two`
> <strong>class선택자</strong>로, 해당 클래스 `<p class="one">` 에게 적용시킵니다. 같은 클래스를 여러 변수들이 가질 수 있습니다. (중복가능)
  
`#three`
> <strong>id선택자</strong>로, 해당 id `<h1 id="three">`는 하나의 변수만이 가질 수 있습니다.  (유일특성)

<br><br>

# 4. 글씨크기

***

```
<head>
    <style>
        .one{font-size:32px;}
        .two{font-size:4em;}
        .three{font-weight:normal;}
        div{font-size:8px;}
    </style>
</head>
<body>
    <p class="one">hello world</p>
    <div>
        <p class="two">hello world</p>
    </div>
    <h1 class="three">hello world</h1>
</body>
```

<p style="font-size:32px">hello world</p>
<p style="font-size:32px">hello world</p>
<p style="font-size:32px">hello world</p>



***


개발자도구 (Ctrl+Shift+i)를 켜서 확인해보면, 세개의 hello world의 폰트크기가 전부 32px임을 확인할 수 있습니다.

`px` 
>고정단위로 크기를 지정해줍니다.

`em`
> 부모의 배수로 크기를 지정해줍니다.   
위의 코드에서는 `div`안에 `class two`가 들어있으므로, `div`의 글자크기 `8px`의 4배를 한 것입니다.

`rem` 
>`body` 글자크기의 배수로 크기를 지정해줍니다. 위의 코드에서는 나오지 않았지만, `body`에서 기본 글자크기는 `16px`이므로, `2rem`을 했다면 `32px`이 나왔을 것입니다.

`h1`
> 글씨크기를 `32px`로 지정하고 `굵게` 설정합니다. 위의 코드에서는 `굵게` 되어 있던 것을 `보통`으로 돌려놓았습니다.



<br><br>

# 5. 박스크기

***

```
<head>
    <style>
        .box-one{
            width:50%;
            height:30vh;
            background-color:green;
        }
        .box-two{
            width:200px;
            height:100px;
            background-color:red;
        }
    </style>
</head>
<body>
    <div class="box-one"></div>
    <div class="box-two"></div>
</body>
```

<style>
        .box-one{
            width:50%;
            height:30vh;
            background-color:green;
        }
        .box-two{
            width:200px;
            height:100px;
            background-color:red;
        }
    </style>
<div class="box-one"></div>
<div class="box-two"></div>

***

박스는 `width`, `height`, `background-color`로 나타낼 수 있습니다.

`px`
> 고정단위로, 크기를 지정해줍니다.

`%`
> 부모가 가진 크기에 대한 비율입니다. `width`는 가능하지만, `height`에 `%`를 사용하면 에러가 발생합니다. 

`vh`, `vw`
>  보이는 화면을 기준으로 `%`를 적용한 것입니다. `width`와 `height` 둘 다 사용이 가능합니다.



<br><br>

# 6. 여백

***
```
<head>
    <style>
        h1{
            border: solid 10px black;
            padding: 20px;
            margin: 20px;
        }
    </style>
</head>
<body>
    <h1>hello world</h1>
</body>
```
<h1 style="border: solid 10px black; padding:20px; margin: 20px">hello world</h1>

***
 
`padding`
> 내부 여백입니다.  
테두리 안쪽의 여백을 만들고 있습니다. <br>  
방향 별로 다르게 여백을 줄 수 있습니다.  
`padding: 10px 20px` (상하 10px, 좌우 20px)  
`padding: 5px 10px 15px 20px`  
(상5px, 우10px, 하15px, 좌20px) <br>  
`padding-top: 5px`  
`padding-right: 10px`  
`padding-bottom: 15px`  
`padding-left: 20px`   
이런 식으로도 여백을 줄 수 있습니다. 

`margin`
> 외부 여백입니다.  
 테두리 바깥의 여백을 만들고 있습니다. <br><br> 
방향 별로 다르게 여백을 줄 수 있습니다.  
`padding`과 마찬가지입니다.
