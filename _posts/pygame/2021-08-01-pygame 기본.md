<h1> 1. 기본 초기화 </h1>

<`.py` 파일을 만들어서 작성하였습니다!>

```
import pygame

pygame.init() # 초기화

# 화면 크기 설정
screen_width = 480 # 가로 크기
screen_height = 640 # 세로 크기
screen = pygame.display.set_mode((screen_width, screen_height))

# 화면 타이틀 설정
pygame.display.set_caption("게임 이름") 

# FPS
clock = pygame.time.Clock()

# 이벤트 루프
running = True # 게임이 진행중인가?
while running:
    dt = clock.tick(60) # 게임화면의 초당 프레임 수를 설정
 
    for event in pygame.event.get(): # 어떤 이벤트가 발생 하였는가?
        if event.type == pygame.QUIT: # 창이 닫히는 이벤트가 발생 하였는가?
            running = False # 게임이 진행중이 아님

    pygame.display.update() # 게임화면을 지속적으로 다시 그리기!

# pygame 종료
pygame.quit()
```
여기까지는 기본적으로 설정해주시면 됩니다. <br>
FPS는 frame per second 의 약자로, 부드럽게 움직이는 정도라고 생각하시면 되겠습니다.<br>
위 코드에서 FPS는 아직 쓰이지는 않았는데, 캐릭터가 움직일 때 사용할 예정입니다.
<br><br>

<h1>2. 게임 속성들 초기화</h1>

캐릭터, 배경 화면, 게임 이미지, 좌표, 속도, 폰트 등을 만들어 보겠습니다.


```
# 배경 이미지 불러오기
background = pygame.image.load("C:/Projects/Python game/pygame_basic/background.png")

# 캐릭터 불러오기
character = pygame.image.load("C:/Projects/Python game/pygame_basic/character.png")
character_size = character.get_rect().size # 이미지의 크기를 구해옴
character_width = character_size[0] # 캐릭터의 가로 크기
character_height = character_size[1] # 캐릭터의 세로 크기
character_x_pos = (screen_width / 2) - (character_width / 2) # 화면 가로 절반 크기에 해당하는 곳에 위치
character_y_pos = screen_height - character_height # 화면 세로 크기 가장 아래에 해당하는 곳에 위치


# 적 enemy 캐릭터
enemy = pygame.image.load("C:/Projects/Python game/pygame_basic/enemy.png")
enemy_size = enemy.get_rect().size # 이미지의 크기를 구해옴
enemy_width = enemy_size[0] # 캐릭터의 가로 크기
enemy_height = enemy_size[1] # 캐릭터의 세로 크기
enemy_x_pos = (screen_width / 2) - (enemy_width / 2) # 화면 가로 절반 크기에 해당하는 곳에 위치
enemy_y_pos = (screen_height / 2) - (enemy_height / 2) # 화면 세로 크기 가장 아래에 해당하는 곳에 위치


# 폰트 정의
game_font = pygame.font.Font(None, 40) # 폰트 객체 생성 (폰트, 크기)
```
캐릭터와 enemy를 만들어 주고, 위치를 설정해주었습니다. <br>
위치는 캐릭터의 왼쪽 위쪽 꼭짓점을 기준으로 하기에, 가운데에 위치하기 위해 저러한 계산이 나온 것입니다. <br>
```
while running:
    screen.blit(background, (0, 0))
    screen.blit(character, (character_x_pos, character_y_pos))
    screen.blit(enemy, (enemy_x_pos, enemy_y_pos))

```
배경/ 캐릭터/ 폰트를 정의해 주었으니, 이것을 `while running`안에 넣어주겠습니다. <br>
전에 있던 코드를 지울 필요 없이 추가만 해주시면 됩니다.<br>

<h1>3. 게임 캐릭터 위치 정의</h1>

```
# 이동할 좌표
to_x = 0
to_y = 0

# 이동 속도
character_speed = 0.6
```
이동할 좌표와 이동 속도를 설정해 주었습니다.<br>
이동속도는 FPS에 따라 변화를 주겠습니다.

```

while running:
    for event in pygame.event.get(): # 어떤 이벤트가 발생 하였는가?
        if event.type == pygame.QUIT: # 창이 닫히는 이벤트가 발생 하였는가?
            running = False # 게임이 진행중이 아님

        if event.type == pygame.KEYDOWN: # 키가 눌러졌는지 확인
            if event.key == pygame.K_LEFT: # 캐릭터를 왼쪽으로
                to_x -= character_speed
            elif event.key == pygame.K_RIGHT: # 캐릭터를 오른쪽으로
                to_x += character_speed
            elif event.key == pygame.K_UP: # 캐릭터를 위로
                to_y -= character_speed
            elif event.key == pygame.K_DOWN: # 캐릭터를 아래로
                to_y += character_speed

        if event.type == pygame.KEYUP: # 방향키를 떼면 멈춤
            if event.key == pygame.K_LEFT or event.key == pygame.K_RIGHT:
                to_x = 0 
            elif event.key == pygame.K_UP or event.key == pygame.K_DOWN:
                to_y = 0 

    character_x_pos += to_x * dt
    character_y_pos += to_y * dt

    # 가로 경계값 처리
    if character_x_pos < 0:
        character_x_pos = 0
    elif character_x_pos > screen_width - character_width:
        character_x_pos = screen_width - character_width
    # 세로 경계값 처리
    if character_y_pos < 0:
        character_y_pos = 0
    elif character_y_pos > screen_height - character_height:
        character_y_pos = screen_height - character_height
```
`while running` 안에 위의 코드를 넣어주겠습니다. <br>
캐릭터의 포지션을 fps에 맞추어 빠르게 움직이도록 하겠습니다.<br><br>
경계값을 처리하지 않으면, 캐릭터가 화면 밖으로 나가버립니다.<br>
이를 방지하고자 화면 범위 밖으로 나가면 위치를 고정했습니다.
<br><br>


<h1> 4. 충돌 처리 </h1>

```

while running:
    # 충돌 처리를 위한 rect 정보 업데이트
    character_rect = character.get_rect()
    character_rect.left = character_x_pos
    character_rect.top = character_y_pos

    enemy_rect = enemy.get_rect()
    enemy_rect.left = enemy_x_pos
    enemy_rect.top = enemy_y_pos

    # 충돌 체크
    if character_rect.colliderect(enemy_rect):
        print("충돌했어요")
        running = False
```
`while running` 안에 코드를 적어주겠습니다. <br>
충돌 처리를 위해서 캐릭터와 enemy의 위치를 설정해주어야 합니다.<br>
위치가 파악되었으니, `if` 문으로 충돌한다면 `running= False` 로 실행이 끝나게 설정해두었습니다. <br><br> 

<h1> 5. 타이머 그리기 </h1>

```
# 총 시간
total_time = 10

# 시작 시간
start_ticks = pygame.time.get_ticks() # 시작 tick 을 받아옴 
```
총 시간과 시작 시간을 설정해 줍니다. <br>
만일 총 시간이 지나면 게임이 종료되도록 만들겠습니다.


```
while running:
        
    # 타이머 집어 넣기
    # 경과 시간 계산
    elapsed_time = (pygame.time.get_ticks() - start_ticks) / 1000 
    # 경과 시간(ms)을 1000으로 나누어서 초 단위로 표시

    timer = game_font.render(str(int(total_time - elapsed_time)), True, (255, 255, 255))
    # 출력할 글자, True, 글자 색상

    screen.blit(timer, (10, 10)) # 타이머를 띄워줍니다.

    # 만약 시간이 0 이하이면 게임 종료
    if total_time - elapsed_time <= 0:
        print("타임아웃")
        running = False
```
`while running` 안에 코드를 넣겠습니다. <br>
타이머의 시간에는 (총시간-경과시간)을 띄워주겠습니다. <br>
타이머 시간이 0 이하라면 종료됩니다.<br>
