## branch 를 생성하고, 생성한 branch에 폴더를 업로드 해보겠습니다

***

```
git branch test1
```
'test1' 이라는 branch 를 생성합니다


```
git checkout test1
```
'test1' branch 로 이동합니다


```
git add webscraping_basic
```
'webscraping_basic' 이라는 폴더를 깃허브에 올리기 위해 `add` 합니다. 각자 원하는 파일명 혹은 폴더명을 입력하시면 됩니다.

```
git commit -m 'test1'
```
`add`한 폴더(파일)을 `commit` 합니다. `-m`을 통해 메시지를 입력할 수 있습니다. <br>
저는 'test1'이라고 입력했지만, 아무렇게나 입력하셔도 괜찮습니다. 중요하지 않습니다.

```
git push origin test1
```
아까 생성한 `test1` branch에 `push` 해줍니다. <br>

이것으로 branch를 새로 생성하고, 생성한 branch에 폴더(파일)를 올리는 것까지 마쳤습니다
