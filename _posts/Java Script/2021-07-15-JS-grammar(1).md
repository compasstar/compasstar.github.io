# 1. JS 코드 입력하기

***

```
<html>
<head>
  <meta charset="UTF-8">
  <title>Document</title>
</head>
<body>
  <p id="one">hello world</p>
  <script>
    document.getElementById('one').innerHTML = 'paullab';
    document.write('hello world write');
    window.alert('hello world alert');
    console.log('hello world console');
  </script>
</body>
</html>
```
* `<script>`부분에 코드를 작성합니다.  
* `<script>`가 꼭 `<body>` 안에 있을 필요는 없지만, 너무 위쪽에서 작성하게 되면 변수 등을 못 읽어올 수 있으니 `<body>`의 맨 아래 쪽에 작성하겠습니다.  
* 앞으로 따로 언급이 없다면 `<script>`안에 코드를 작성했다고 생각해주시면 됩니다. <br>
* 페이지를 보고 싶다면, 코드를 입력한 후 `Ctrl`+`Shift`+`Q` 를 누르시면 됩니다.

# 3. 연산

***

```
var x, y;
x=3;
y=5;

document.write(x+y);
document.write(x-y);
document.write(y / x);
document.write(y % x);
document.write(x * y);
document.write(x ** y);
```
***
변수 간에 연산이 가능합니다. <br>
각각 8, -2, 1.666667, 2, 15, 243 이 출력되게 됩니다.<br>
`/`는 나누기, `%` 는 나머지, `*`는 곱셈, `**`는 제곱을 뜻합니다.

```
x++;
y--;
x *= 3
```
`++`는 1만큼 더하라는 뜻입니다. `x`를 출력한다면 4가 출력됩니다. <br>
`--`는 1만큼 빼라는 뜻입니다. `y`를 출력한다면 4가 출력됩니다.<br>
`x *= 3`은 `x = x * 3`과 같은 뜻입니다. `+` `-` `/` 등도 모두 가능합니다.



<br><br>

# 4. 등호

***

```
var x, y, z;
x=3;
y=5;
z='3';
document.write('1: ',x > y);
document.write('2: ',x >= y);
document.write('3: ',x < y);
document.write('4: ',x <= y);
document.write('5: ',x == z); 
document.write('6: ',x != y);
document.write('7: ', x === z);
```

***

등호를 통한 비교가 가능합니다. <br>
3, 4, 5, 6 은 `true` 1, 2, 7 은 `false`가 나타납니다. <br>
`==`는 같다, `!=`는 같지 않다를 뜻합니다. <br>
`==`은 타입을 고려하지 않고 비교하지만, `===`은 타입까지 고려하여 비교합니다. <br>(7번은 숫자와 문자열을 비교하기에 `false`)


<br><br>

# 5. 비교연산자

***

```
var x, y;
x = true; 
y = false;
document.write('1: ',x && y);
document.write('2: ',x || y);
document.write('3: ', !y);
```

***
1: `false` 2: `true` 3: `true` 입니다.

`&&`
> and를 나타냅니다. (`true` and `false`) -> `false`

`||`
> or를 나타냅니다. (`true` or `false`) -> `true` <br>
> 키보드에서 Enter 위에 '원'표시를 Shift해서 누르시면 나옵니다.

`!`
> not을 나타냅니다. (not `false`) -> `true`

# 9. 배열

***
```
var 배열 = [10, 20, 30, 40, 50];
```

```
document.write(배열[1]);
```
20
***
해당 인덱스을 출력합니다. <br><br>
```
document.write(배열.slice(1,4));
```
20, 30, 40
***
인덱스 1이상 4미만을 출력합니다. <br><br>
```
배열.pop();
```
50
***
마지막 인덱스를 빼고, 리턴합니다. <br><br>
```
배열.push(1000);
```

```
document.write(배열.join('!!'));
```
10!!20!!30!!40!!1000
***
배열의 사이들에 값을 끼워넣습니다. <br><br>
```
배열.sort();
```
10,1000,20,30,40
***
배열을 정렬합니다. <br><br>
```
배열.reverse();
```

# 10. 딕셔너리

***
```
var person = {
    name:{
    firstname:'hojun',
    lastname:'lee',
    },
    age:10,
    married:true
};
```
***
`{}`를 이용하여 딕셔너리를 만들 수 있습니다.  
`hojun`을 얻고 싶다면 `person['name']['firstname']`으로 찾을 수 있습니다.  
`age`를 얻고 싶다면 `person['age']`으로 찾을 수 있습니다.
