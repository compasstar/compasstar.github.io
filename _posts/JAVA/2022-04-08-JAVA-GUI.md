간단한 GUI 프로그램을 만들어보겠습니다

```
import javax.swing.*;
```
'swing'을 사용하여 만들기 때문에, import 해주세요<br><br>

```
public static void main(String[] args) {
	JFrame window = new JFrame();

	//settings for the frame;
	window.setSize(300, 300);
	window.setTitle("First GUI");
	window.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
	window.setVisible(true);
}
```
프레임을 생성하고 간단한 설정을 마치면 GUI를 생성할 수 있습니다. <br>
`window.setVisible(true)`는 코드의 마지막에 들어가야 합니다. 중간에 버튼을 생성하는 등의 코드를 넣고 싶다면 `window,setVisible(true)`보다 위쪽에 적어주셔야 합니다. <br><br>

```
JButton button1 = new JButton();
JButton button2= new JButton();
JButton button3= new JButton("Explore");
JButton button4= new JButton("Cancel");
JButton button5= new JButton("Send");

//settings for button object;
button1.setText("Back");
button1.setBackground(Color.CYAN);
		
button2.setText("Next");
button2.setBackground(Color.RED);
```
버튼을 생성하고, 색깔을 입혀줍니다.

```
window.setLayout(new BorderLayout());
		
window.add(button1, BorderLayout.EAST);
window.add(button2, BorderLayout.WEST);
window.add(button3, BorderLayout.CENTER);
window.add(button4, BorderLayout.NORTH);
window.add(button5, BorderLayout.SOUTH);
```
레이아웃을 `BorderLayout`으로 설정하고, 생성한 버튼들을 위치에 맞추어 추가해주겠습니다. `BorderLayout`은 버튼들을 방향에 따라 프레임을 꽉 채워줍니다.<br><br>

```
button1.addActionListener(new FirstGUI().new changeColor());	

private class changeColor implements ActionListener{

	@Override
	public void actionPerformed(ActionEvent e) {
		//code to do when you click a button
		System.out.println("Clicked");
	}
}
```
버튼에 이벤트를 만들 수 있습니다. 이벤트 역할을 하는 클래스를 생성한 뒤, 버튼에 이벤트리스너를 추가해줍니다. 위 코드같은 경우 클릭을 하면 "CLicked"가 출력됩니다.<br><br>

***
### 전체코드

```
import java.awt.BorderLayout;
import java.awt.Color;
import java.awt.FlowLayout;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

import javax.swing.*;

public class FirstGUI {
	
	public static void main(String[] args) {
		JFrame window = new JFrame();
		//settings for the frame;
		window.setSize(300, 300);
		window.setTitle("First GUI");
		window.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
		
		
		JButton button1 = new JButton();
		JButton button2= new JButton();
		JButton button3= new JButton("Explore");
		JButton button4= new JButton("Cancel");
		JButton button5= new JButton("Send");
		//settings for button object;
		button1.setText("Back");
		button1.setBackground(Color.CYAN);
		
		button2.setText("Next");
		button2.setBackground(Color.RED);
		
		//BorderLayout layout = new BorderLayout();
		
		window.setLayout(new BorderLayout());
		
		window.add(button1, BorderLayout.EAST);
		window.add(button2, BorderLayout.WEST);
		window.add(button3, BorderLayout.CENTER);
		window.add(button4, BorderLayout.NORTH);
		window.add(button5, BorderLayout.SOUTH);
		
		button1.addActionListener(new FirstGUI().new changeColor());
		
		
		
		window.setVisible(true);
		
	}
	
	private class changeColor implements ActionListener{

		@Override
		public void actionPerformed(ActionEvent e) {
			//code to do when you click a button
			//change();
			System.out.println("Clicked");
		}
	}
}
```
