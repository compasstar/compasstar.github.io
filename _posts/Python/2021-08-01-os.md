```
import os
```
os 모듈을 받아주겠습니다.

```
current_path = os.path.dirname(__file__) 
image_path = os.path.join(current_path, "images")
```

`os.path.dirname(__file__)`
> 현재 파일의 위치를 반환합니다.

`os.path.join(current_path, "image")`
> current_path 안의 "image" 위치를 반환합니다.
