# 2. table

***


```
dq1 <- table(admin$dq1) # 테이블 채로 저장
prop.table(dq1) # 비율
```

table(admin$dq1)

# 3. 결측값 관리

***


```
life <- read.csv(file="life.csv", na.strings="99")
```
'99' 값을 NA로 바꾸어서 파일을 부릅니다

```
table(life$v5, useNA = "ifany") 
```
NA가 있다면 따로 표기합니다 <br>

```
mean(v5, na.rm = T) 
```
결측값을 제외하고 평균을 구합니다
