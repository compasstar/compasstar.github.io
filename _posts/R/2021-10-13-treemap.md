<h2>R의 패키지 중 하나인 treemap 에 대해 알아보겠습니다</h2>

### (사용한 데이터들은 모두 R에 원래 있던 내장데이터들입니다)

<br><br>

# 1. treemap 설치하기

***

```
install.packages("treemap")
library(treemap)
```
`search()`를 입력하면 `library`의 패키지들을 확인할 수 있습니다. <br>
`treemap` 이 설치되었는지 확인하시기 바랍니다.<br>
<i>(그냥 코드 적는 곳에 입력하시면 됩니다)</i> <br><br>

# 2. treemap

***

![treemap](https://user-images.githubusercontent.com/55419868/137078890-406e9acf-53cb-4697-93fe-5bd00ea496aa.png)




```
treemap(state, # state 데이터를 써
        index = c("Name"), # 주이름을 넣어줘
        vSize = "Area", # 면적이 클수록 Area크다
        vColor = "Murder", # 색깔 진할수록 Murder 크다
        type = "value", # 컬러링 방법
        title = "The US Murder") # 제목
```
