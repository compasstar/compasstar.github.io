# 2. 산점도 그래프

***

```
ggplot(data=Orange, aes(age, circumference))+
  geom_point(color="Red") # 점의 색깔 변경
```

`ggplot() + ` 형태로 추가할 것들을 붙여주며 그래프를 완성해나갑니다. <br>
`geom_point()` 는 산점도를 그리라는 뜻입니다. <br>

```
ggplot(data=Orange, aes(age, circumference))+
  geom_point()+
  stat_smooth(method = 'lm') # 추세선(회귀선)을 그어라
```

```
ggplot(data=Orange, aes(Tree, circumference, color=Tree))+
  geom_point()
```

`aes(color=Tree)` Tree 에 따라 점의 색깔을 다르게 합니다.<br><br>


# 3. 막대 그래프

***

```
ggplot(data=Orange, aes(Tree, circumference))+
  geom_bar(stat="identity", # 일반 막대그래프
           fill="orange") # 막대 색깔을 채워라
```
`geom_bar(stat="identity")` 로 막대그래프를 그립니다. <br><br>

```
ggplot(data=iris, aes(x=Species))+
  geom_bar(fill = 'salmon')+
  ggtitle("Frequency by Species") # title을 그립니다.
```

```
ggplot(data=iris, aes(x=Species))+
  geom_bar(fill = 'salmon')+
  coord_flip() # 가로 막대그래프
```

# 4. 히스토그램

***

```
ggplot(cars, aes(x=dist))+ 
  geom_histogram(fill="skyblue",
                 color="blue")
```
원래 `data=cars` 이지만 `data=`는 생략해서 써도 상관없습니다.<br>
`fill`은 막대의 색을 칠합니다. <br>
`color`는 막대 테두리의 색을 칠합니다. <br>

```
ggplot(data=iris, aes(x=Petal.Length, fill=Species))+
  geom_histogram(position="identity",
                 binwidth = 0.3) # 막대의 넓이
```
`fill = Species` -> Species 에 따라 막대의 색을 칠합니다. <br>

`position`
> stack -> default 값으로, 누적해서 막대 위로 그려줍니다.<br>
> identity -> 그대로 그래프를 겹쳐 그립니다.<br>
> dodge -> 그래프를 옆으로 분리해서 그립니다.<br>
> fill -> 1을 기준으로 비율을 계산해서 채웁니다.<br>

# 5. 원그래프

```
# 원그래프에 사용할 데이터프레임을 만듭니다
species <- c("setosa", "versicolor", "virginica")
freq <- c(50, 50, 50)
a <- data.frame(species, freq)

ggplot(a, aes(x="species", y=freq, fill=species))+
  geom_bar(stat="identity")+ # 막대를 만들어줘
  coord_polar(theta="y")+ 
  geom_text(aes(label=paste0(round(freq/1.50,2), "%")),
            position = position_stack(vjust=0.5))
```
원그래프는 우선 누적막대그래프를 그린 후, `coord_plot(theta="y")` 를 이용하여 원그래프로 만들어 줍니다. <br>

`x="species"`
> 따옴표를 붙이면 누적막대그래프를 그리기 위해, 누적으로 자료를 만들어줍니다. <br>
> 
`aes(label=)`
> 그래프 위에 라벨을 붙여줍니다. <br>

`position = position_statck(vjust=0.5)` 
>글씨를 그래프의 중앙으로 가져옵니다. <br>

<br>

`paste0` / `paste`의 차이
> `paste(...)` 는 사실 `paste(..., sep=' ', collapse=NULL)` 입니다. <br>
> `sep` 는 붙이고자 하는 두 요소 간에 이어주는 문자로, 기본값은 띄워쓰기 입니다. <br>
> `collapse` 는 붙여진 결과끼리 서로 이어주는 문자로, 기본값은 NULL 입니다. <br>
> 예시 <br>
> paste(rep('bring',4), rep('book',4), sep=" a ", collapse=", ") <br>
> 출력 <br>
> "bring a book, bring a book, bring a book, bring a book"<br><br>
> `paste0` 는 paste(..., sep='') 을 뜻합니다. 즉, 띄워쓰기를 없애줍니다. <br>

# 7. 선그래프

***

```
ggplot(data=economics, aes(date, unemploy))+
  geom_line(size=2, # 선 굵기
            lty=1, # 선 타입 line type
            color="gold")+
  ggtitle("unempolyment")+
  geom_hline(yintercept=mean(economics$unemploy),
             lty=2) # 평균선을 넣어라
```
