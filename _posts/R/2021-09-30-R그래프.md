## 3. barplot

<img width="300px" src="https://user-images.githubusercontent.com/55419868/135460644-101d250a-5b47-4234-852e-3539bda3406d.png">

```
survive.by.class <- margin.table(Titanic, margin = c(4,1))
barplot(survive.by.class,
        beside= T, # 수평배열: 왼쪽오른쪽으로 막대 나누기
        col = c("black", "green"), # color
        legend = T, # 범례
        names.arg = c("1등석", "2등석", "3등석", "선원"))
```
`beside=T` 가 없다면 검은색과 녹색의 막대가 하나로 그려집니다
<br><br>

<img width="300px" src="https://user-images.githubusercontent.com/55419868/135460693-08bd589c-fa18-426e-9896-70aa5a07a7e8.png">


```
barplot(head(state$Income))
axis(1, at=1:6,
     lab=c("Alba","Alas","Arizo","Arkan","Cali","Colo"))
```
(이미지에서는 잘렸지만, 6개의 이름이 전부 붙여져 있습니다)<br>

만일 `horiz=T` 를 입력하면 그래프가 수평으로 세워집니다. 즉, 왼쪽에서부터 뻗어나오는 그래프를 볼 수 있습니다.

## 5. 선그래프

<img width="300px" src="https://user-images.githubusercontent.com/55419868/135460824-c6f26322-665f-4a7a-bb43-73d38bce4863.png">


```
# 데이터를 만들어 줍니다
x <- c(100, 200, 180, 150, 160)
y <- c(220, 300, 280, 190, 240)
z <- c(310, 330, 320, 290, 220)

# 우선 x데이터 만으로 plot 을 만들어줍니다
plot(x,
     type="o", col="red",
     ylim=c(0,400), axes=F, ann=F)

# x축을 5개의 범위로 나누고, y축 범위를 설정해줍니다
# 1은 x축, 2는 y축
axis(1, at=1:5, 
     lab=c("A","B","C","D","E")) 
axis(2, ylim=c(0,400))

# y, z 선을 추가해줍니다
lines(y, type="b", pch=17, col="green", lty=5)
lines(z, type="b", pch=11, col="blue", lty=5)

# 제목을 붙여줍니다
title(main="book page",
      col.main="purple", # 제목의 색깔
      xlab="book", ylab="Page",
      col.lab="gray") # 라벨의 색깔

# 범례를 만듭니다
legend(4.5, 400, # 좌표 위치 (4.5, 400)
       c("sci", "Eng", "math"), # 각 선의 이름
       cex=1.0, 
       col=c("red", "green", "blue"),
       pch=21, 
       lty=1) 
```

`type = "o"` -> 점과 선의 형태 <br>
`ylim = c(0,400)` -> y축의 범위는 0부터 400까지 <br>
`axes = F` -> x,y 축의 테두리를 없게 합니다 <br>
`ann = F` -> 제목을 없게 합니다 <br>
`pch` -> [point character] 점의 모양으로 나타내는 것으로, 0~25번까지 다양합니다. <br>
`lty` -> [line type] 선의 종류를 나타내는 것으로, 1~6번까지 있습니다 <br>
`cex` -> 글자 크기를 조절합니다
<br><br>
