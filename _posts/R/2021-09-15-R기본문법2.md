## 10. sort 정렬

```
x <- c(3,6,4,8,2,9,7)
sort(x) # 2 3 4 6 7 8 9
sort(x, decreasing=T) # 내림차순으로 정렬 9 8 7 6 4 3 2
sort(x, decreasing=F) # 오름차순으로 정렬 2 3 4 6 7 8 9
```

## 11. apply

```
apply(score,1,sum) # 각 행의 sum 을 구해라
apply(score,1,mean) # 각 행의 mean 을 구해라
apply(score,2,sum) # 각 열의 sum 을 구해라
apply(score,2,mean) # 각 열의 mean 을 구해라
```
1은 행을 가르키고, 2는 열을 가르킵니다. <br><br>

## 12. DataFrame

```
x1 <- c(1,2,3,4)
x2 <- c(90,80,70,60)
a <- data.frame(x1,x2)

dim(a) # 4 2 : 데이터프레임의 행렬 크기를 의미한다 4행 2열
View(a) # 새 창에 데이터프레임을 띄워서 봅니다

a$x3 <- c(900,910,920,930) # 열을 추가합니다.
```

```
  x1 x2
1  1 90
2  2 80
3  3 70
4  4 60
```
데이터프레임에서는 `$`를 사용하여 항목을 선택할 수 있습니다. <br>
위의 예시와 같이 `x3`가 없었다면 새로 추가합니다.

## 15. 반복문 for, while, repeat

```
for(i in 3:7){
  x <- 10-i
  print(x)
}

i <- 1
while(i<=10){
  print(i)
  i <- i+1
}

i <- 1
repeat{
  print(i)
  if(i>=10){
    break
  }
  i <- i+1
}
```
`repeat` 문은 무조건 한 번 이상 실행하며, 계속 반복하기에 `break`를 통해 반복을 중지할 필요가 있다.

## 16. 함수 function

```
abc <- function(a,b,c){
  answer <- a+b+c
  return(answer)
}

abc(10,30,50) # 90
abc(3,7,90) # 100
```
