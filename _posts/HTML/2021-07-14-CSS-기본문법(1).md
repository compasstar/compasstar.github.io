CSS는 페이지를 꾸미는 역할을 합니다. 예를들어 글씨 색을 바꾼다거나, 가운데 정렬을 한다거나 할 수 있습니다.

구름 IDE를 이용했음을 미리 밝힙니다.

단축키는 명령어를 입력하고 Tap을 입력하면 됩니다. 

> h + Tap --> `<h1></h1>`

<br>
<br>

# 1. CSS 코드 입력하기

***

<h1 style="color:red;">hello world</h1>

hello world를 붉은색으로 꾸미는 코드를 어디에 입력해야 할까요?


```
<head>
    <style>
        h1{color:red;}
    </style>
</head>
<body>
    <h1>hello world</h1>
</body>
```
첫번째 방법은 HTML파일의 `<head>`에 `<style>`로 작성하는 것입니다. 

```
<h1 style="color:red;">hello world</h1>
```
두번째 방법은 `<h1>`에 직접 `<style>`을 적어주는 것입니다.
```
<head>
    <link rel="stylesheet" href="001.css">
</head>
<body>
    <h1>hello world</h1>
</body>
```
```
h1{color:red;}
```
세번째 방법은 .css파일을 따로 만들어 link를 걸어주는 것입니다.  
위의 코드는 .html 파일이고, 아래 코드는 .css 파일입니다.  
두 파일을 link로 연결시켜주면 됩니다.  
단축키는 `link` + `Tap` 입니다.  

앞으로는 첫번째 방법을 사용해서 설명하겠습니다.


<br><br>

# 2. 명칭

***
```
<style>
    h1, h2, h3{
        color:skyblue;
    }
</style>
```
<h1 style="color:skyblue;">hello world</h1>
<h2 style="color:skyblue;">hello world</h2>
<h3 style="color:skyblue;">hello world</h3>

***

`<style>`을 기준으로  
h1, h2, h3 을 <strong>선택자(Selector)</strong>라고 합니다.  
color 을 <strong>속성(Property)</strong>라고 합니다.  
skyblue 을 <strong>값(Value)</strong>라고 합니다.



<br><br>

# 3. 선택자

***

```
<head>
    <style>
        * {font-weight: bold;}
        h2 {font-size: 4px;}
        .one {color:red;}
        .two {border: solid 1px black;}
        #three {font-style: italic;}
    </style>
</head>
<body>
    <h1 id="three">hello world</h1>
    <p>hello world</p>
    <p class="one two">hello world</p>
    <p class="one">hello world</p>
    <h2>hello world</h2>
</body>
```

![css1](https://user-images.githubusercontent.com/55419868/125548557-6bf9f5bc-1f92-41e9-b882-27b0ebb2ba2f.JPG)

***

`*`는 <strong>범용선택자</strong>로, 모두에게 적용시킨다.  
`h2`는 <strong>타입선택자</strong>로, h2에게 적용시킨다.  
`.one`, `.two`는 <strong>class선택자</strong>로, 해당 클래스 `<p class="one">` 에게 적용시킨다. 같은 클래스를 여러 변수들이 가질 수 있다. (중복가능)  
`#three`는 <strong>id선택자</strong>로, 해당 id `<h1 id="three">`는 하나의 변수만이 가질 수 있다.  (유일특성)

<br><br>

# 4. 크기

***

```
<head>
    <style>
        .one{font-size:32px;}
        .two{font-size:2rem;}
        .three{font-weight:normal;}
        div{font-size:8px;}
        .box-one{
            width:50%;
            height:30vh;
            background-color:green;
        }
        .box-two{
            width:200px;
            height:100px;
            background-color:red;
        }
    </style>
</head>
<body>
    <p class="one">hello world</p>
    <div>
        <p class="two">hello world</p>
    </div>
    <h1 class="three">hello world</h1>
    <div class="box-one"></div>
    <div class="box-two"></div>
</body>
```

<p style="font-size:32px">hello world</p>
<p style="font-size:32px">hello world</p>
<h1 style="font-weight:normal">hello world</h1>


***


개발자도구단축키 Ctrl+Shift+i  
px 고정단위  
em 부모의 배수  
rem body의 배수  
width:50%; 부모가 가진 width의 50% (vw도있음)  
heigth: %불가, 30vh 보이는화면의 30%

# 6. 이미지

***



```
 <img width="200px" height="auto" src="a.JPG" alt="hello">
```

![apple-icon-144x144](https://user-images.githubusercontent.com/55419868/125251803-a9b4cb00-e332-11eb-921a-bac7cf4df10f.png)

***

`width`만 설정을 한다면 `height`는 자동으로 `auto`를 갖습니다. (이미지의 비율을 유지합니다)

그러나 둘 다 설정을 한다면, 원본 이미지의 비율이 깨집니다.

`alt`는 이미지가 없을 시, 대체할 문구입니다. 만약 `a.JPG`가 존재하지 않는 이미지라면 "hello"를 출력합니다.
