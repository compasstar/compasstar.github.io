<h3>HTML 코드</h3>

```
<head>    
    <meta property="og:image" content="" />
    <meta property="og:url" content="" />
</head>
```

먼저 HTML부터 보겠습니다. `<head>`에 `meta`를 입력하는 부분이 있습니다. 이 코드에 의하여 검색할 때 노출이 결정됩니다. <br><br>

`contents`가 비어있는 것을 볼 수 있습니다.<br> 이 부분에 자바스크립트를 통해서 image와 현재사이트의 url을 넣어보겠습니다. <br><br>


<h3>JS 코드</h3>

```
$(function(){
    let url = window.location.href;
    let img = $('img').attr('src');

    $("meta[property='og\\:url']").attr('content', url)
    $("meta[property='og\\:image']").attr('content', image)
});
```

url과 img를 먼저 가져옵니다.

`window.location.href`
> 현재 사이트의 url입니다.

`$('img').attr('src')`
> img의 주소를 가져옵니다.

`$("meta[property='og\\:url']").attr('content', url)`
> `property`값으로  `'og:url'`을 갖는 `meta`를 불러와서 `content`에 `url`을 넣어줍니다.
