## 1. heapify

```
def heapify(keys, idx, n):
        left = idx*2
        right = idx*2 +1
        p_idx = idx
        
        if (left <= n and keys[p_idx] < keys[left]):
            p_idx = left
                
        if (right <= n and keys[p_idx] < keys[right]):
            p_idx = right
                
        if p_idx != idx:
            keys[p_idx], keys[idx] = keys[idx], keys[p_idx]
            return heapify(keys, p_idx, n)
```
힙화 한다는 것은 힙의 형식에 맞춘다는 것을 의미합니다. <br>
> 부모가 자식보다 크기가 크고(max_heap의 경우입니다. min_heap의 경우는 반대), 완전이진트리여야 합니다.

그렇기에 위 함수는 어떠한 원소가 주어졌을 때, 자식들과 크기를 비교해 나가며 부모가 자식보다 크기가 크도록 위치를 바꾸어 나가는 코드입니다. <br><br>
만약 min_heap (부모가 자식보다 크기가 작은 힙) 인 경우는 위에서 부등호의 방향만 바꾸어 주면 됩니다. <br><br>
이 글에서는 max_heap 기준으로 작성하였습니다.

## 2. insertItem

```
def insertItem(key):
    heap.append(key) # heap 은 원소들이 들어있는 배열
    indexNum = len(heap)-1 # 추가된 키에 해당되는 index

     while indexNum >= 2:
        if heap[indexNum]>heap[indexNum//2]:
            heap[indexNum], heap[indexNum//2] = heap[indexNum//2], heap[indexNum]
            indexNum = indexNum//2
        else:
            break
```
우선 힙에 키를 추가합니다.<br>
 그 후 반복문을 이용하여 크기를 비교해가며 해당 키를 상단으로 올려줍니다. <br>
부모에 해당하는 인덱스는 자식의 인덱스를 2로 나눈 몫에 해당되므로,  `heap[indexNum]>heap[indexNum//2]` 이렇게 자식과 부모를 비교할 수 있습니다.<br>
만약 자식이 부모보다 크다면 둘의 위치를 바꾸어 줍니다.

## 3. removeRoot

```
def removeRoot():      
        if len(heap)-1 > 1:
            heap[1], heap[-1] = heap[-1], heap[1]
            heap.pop(-1)
            heapify(1)
            
        else:
            return
```

루트를 제거하고 힙의 형식에 맞추어 정렬하는 것이 목표입니다. <br>
루트와 마지막 원소를 변경하고 `.pop()`을 통해 원래 루트였던 것을 제거해줍니다. <br>
그렇다면 원래 마지막 원소가 루트 위치에 왔기 때문에, 힙의 형식에 맞지 않습니다.<br>
그렇기에 새로운 루트를 힙화해주어 힙의 형식에 맞추어 줍니다.<br>
`heapify(1)` 1 인 이유는 루트가 index 1번에 있기 때문입니다. 힙에서는 index 0은 비워두고 1부터 루트가 시작됩니다.

<br><br>

## 4. heapSort

```
def heapSort(heap):
    heap = [0] + heap # 0번 인덱스를 비워줍니다.
    n = len(heap)-1
    
    # 힙의 끝부터 차례로 힙화해줍니다
    for idx in range(len(heap)-1,0,-1):
        heapify(heap, idx, n)
               
    # 첫 원소와 마지막 원소를 바꾸어주며 한 칸씩 힙의 범위를 줄여나가며 정렬합니다           
    for idx in range(len(heap)-1, 1, -1):
         heap[idx], heap[1] = heap[1], heap[idx]       
         heapify(heap, 1, idx-1)      
```

첫번째 반복문에서는 마지막 원소부터 차례로 힙화해주면서 힙의 형태를 만듭니다. (max_heap 정렬 된 상태) <br><br>
두번째 반복문에서는 루트와 마지막 인덱스를 바꾸는 것으로 루트를 끝으로 보냅니다. <br>
끝으로 간 루트를 제외하고 작아진 힙을 힙화해주어 형식을 맞추어 줍니다. <br>
점점 힙의 크기를 줄여나가면 min_heap의 형태를 구할 수 있습니다.
